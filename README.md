# oh-my-terminal

个人终端配置环境集合

## 工具集合

* zsh
* tmux
* vim

## 使用说明

1. 安装必要的软件

    ```bash
    sudo apt-get install git zsh tmux vim
    ```

1. 备份必要的配置文件

1. 安装使用

    ```bash
    git clone https://github.com/alvisisme/oh-my-terminal.git ~/.oh-my-terminal
    sh ~/.oh-my-terminal/setup.sh
    ```

## 效果预览

```bash
docker build -t oh-my-terminal .
docker run -it oh-my-terminal
```
